## :hammer:Funcionalidades del proyecto

- `hola.py`: Uso de tkinter para hacer ping con GUI
- `multi.py`: Hacer varios pings usando la libreria ping3
- `ping.py`: Hacer un ping ICMP usando la libreria ping3
